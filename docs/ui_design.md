# AI Code Recognizer — Premium Mobile UI Design (MVP)

This file contains the premium, futuristic mobile UI design brief to be used when building the client for the app.

High-level details are taken from the original requirement specification and condensed into an actionable style guide.

Key points:
- Glassmorphism, frosted glass panels and soft glow neon accents (#3A7BFD, #B446FF, #00C9FF)
- Smooth gradients (blue → indigo → violet), rounded corners (16–24px), high contrast
- Light / Dark themes supported
- Screens: Welcome, Home/Upload, Scanning, Result, History, Settings
- Animations: scanning HUD, rotating gradient ring, micro-interactions, ripple glow

Assets & Next steps:
- Create a Figma/Sketch prototype following these colors and spacing rules
- Provide vector icons for common languages
- Build React Native or Flutter components using glassmorphism backgrounds and SVG animations
